🍔 Food Delivery Analytics Dashboard (Power BI)
📌 Project Overview

This project is an interactive Power BI dashboard designed to analyze and visualize food delivery data. It provides deep insights into revenue, customer behavior, restaurant performance, and delivery efficiency to help businesses make data-driven decisions.

🚀 Key Highlights
💰 Total Revenue: 165M
📦 Total Orders: 200K
🧾 Average Order Value: 824.57
⏱ Average Delivery Time: 44.55 mins

These KPIs are prominently displayed in the dashboard (Page 1) for quick business insights.
